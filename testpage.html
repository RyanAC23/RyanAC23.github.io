    Special characters with &#

    For example, a name with é would use special character 233, preceded by &#.

    This also works for &amp and &quot

    JARGON:

    • tag: commands within < > brackets
    • element: Everything in between the brackets, <LI>like this list element.</LI>
    • empty element: Any element that doesn't affect a block of text, and thus doesn't require closing (p, img)
    • container: A name occasionally used for elements marking blocks of text.
    • attributes: Modifiers to elements, such as width="", which are often optional.
