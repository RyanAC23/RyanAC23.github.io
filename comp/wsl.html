Install WSL


> Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux
> curl.exe -L -o ubuntu-1604.appx https://aka.ms/wsl-ubuntu-1604
> Add-AppxPackage .\app_name.appx

> sudo apt update
> sudo apt upgrade

> sudo apt install gcc gfortran lsb git mercurial
> sudo apt upgrade

Colors


* Download the Microsoft ColorTool here.
* Extract it somewhere, and navigate to it in cmd. Run the following:
> ColorTool.exe -x -d \schemes\[pick a scheme]
* This will change the color theme. I found that this was most of the battle, but the highlighted folders are awful and often unreadable. Get rid of them by adding this to your .bashrc:
> LS_COLORS=$LS_COLORS:'ow=1;34:' ; export LS_COLORS
WSL is now about as usable as it'll ever be. You can set up an xserver and run graphical packages, but it's going to be slow, and at that point you really need to dual boot or ditch your OS.
